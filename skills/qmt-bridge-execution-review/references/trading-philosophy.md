# 交易观标准（操作评价依据）

本文件为 **qmt-bridge-execution-review** 操作评价的规范来源。评价脚本 `execution_review_eval.py` 将下列原则量化为可检查项（统计归纳，非投资建议）。

---

## 一、化繁为简的 A 股框架

1. **四板块**：大金融、消费、周期、科技。
2. **驱动力**（摘要）：
   - **大金融**：顺周期是牛市基础；银行是核心资产支撑；非银看并购与创新（如 2025 并购新规利好券商去同质化）；稳住市场即政策目标。
   - **消费**：人口大国；必选为底，可选看品牌；创新药、品牌中药长期受益老龄化。
   - **周期**：与复苏节奏相关；需和新经济接轨（化工新材料、能源金属、稀土等）；黄金乱世低买。
   - **科技**：弹性最大、投机资金主战场；与科技革命迭代相关（移动互联网、人工智能等）。
3. **轮动**：超大资金偏「大金融+消费」同步；周期看供应链紊乱+复苏；科技看产业革命。
4. **注册制后结构**：二八蓝筹与「涨多跌少」科技股行情交替。

## 二、赛道趋势投资法（执行标准）

| 原则 | 评价中的体现 |
|------|----------------|
| 选赛道、做减法、精选股 | 持仓/交易标的数量、板块分散度 |
| 看长做短，**量能**为重要参考 | 两市成交额分区 + 现金占比是否匹配 |
| 股票池三级：基础→精选→**交易池极少** | 当日有成交标的数、总持仓数 |
| **低吸不追涨，顺势而为** | **以买入均价在当日振幅中的位置**判断：贴近低价区=低吸；贴近高价区=追涨；**收盘大涨若买在低位视为低吸成功** |
| **大跌分步低吸，大涨分步止盈** | 逆势加仓、单日大涨未卖、近 3 日累计涨幅未卖 |
| 单日涨幅偏离大、**连续 3 日**累计一定涨幅须分步止盈 | 阈值见 `trading_philosophy.py` |
| 控制回撤，积累收益；会卖的是师傅 | 盈亏日、滑点、撤单、现金 |
| 交易要算账 | 滑点 bp、当日盈亏拆解 |
| 专注交易时间 | 委托笔数、频繁撤单 |

### 量能参考（亿元，全市场成交额，宜人工校准）

| 区间 | 仓位态度 |
|------|----------|
| **&lt; 1 万亿** | 谨慎区，尽量减少仓位 |
| **1～1.5 万亿** | 适度参与，赛道结构性行情 |
| **≥ 1.5 万亿且能持续** | 强力做多区；不能持续则警惕短期头部 |

> 量能非绝对，需随市场调整；复盘脚本默认从 Bridge 指数 tick 自动汇总两市成交额（亿），也可用 `--market-turnover-yi` 覆盖。

**近 3 日市场热度**（自动）：从指数日 K 的 `amount` 拉取上证+深证近 3 个交易日序列，评估升温/降温/持续高热/放量回落等（需本地已缓存指数日 K，缺失时会触发下载）。

## 三、避险增值的交易戒律

1. **操作与研究分离**：前瞻是预测，交易敬畏市场；无趋势则少交易，**保全本金优先**。
2. **市场永远是对的**：尊重市场，忌主观臆断逆势硬做。
3. **化繁为简**：规律一致、可复盘。
4. **做少数派**：不从众，成功判断往往是少数派。
5. **逆向思维**：别人贪婪时恐惧，别人恐惧时贪婪（用于评价「大跌加仓」是否理性分步，而非鼓励抄底）。

## 四、与脚本阈值对应

见 `skills/_shared/trading_philosophy.py`：

- 单日大涨分步止盈：`pct_chg ≥ 5%` 且卖出不足
- 近 3 日累计止盈：`3 日累计 ≥ 8%` 且卖出不足
- 追涨：买均处于振幅上段（≥65%）、或买均贴近收盘价（≥收盘×99.5%）
- 低吸：买均处于振幅下段（≤40%）、或 **收盘较买均上行 ≥2%**（买在后段仍低于收盘，收盘大涨算低吸成功）
- **不用**收盘涨跌幅单独定罪追涨
- 逆势：`当日 ≤ -2%` 净买入
- 交易池：持仓 &gt; 6 或当日成交标的 &gt; 4 提示做减法
- 滑点：`≥ 30bp` 提示精打细算；`≥ 100bp` 严重

Agent 生成文字复盘时，应优先引用评价输出中的 **「交易观对照」** 与 **「戒律检查」** 小节。
